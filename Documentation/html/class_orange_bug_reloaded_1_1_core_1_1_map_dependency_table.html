async Task OrangeBugReloaded.Core.MapDependencyTable.DoAsyncWorkFollowingDependenciesAsync ( IEnumerable< Point >  initialBag,
Func< Point, Task< bool?>>  action 
)

Executes an async action on a set of points with respect to dependencies. That means that points and their dependencies (and further transitive dependencies) are visited in the correct order.

Parameters
initialBagThe points where action must be executed on.
actionAn action that is executed for each visited point. Depending on the changes made by the action, this action might also be executed multiple times for the same point. A return value of null indicates that the whole algorithm should be stopped and no other points should be visited.
