virtual Task OrangeBugReloaded.Core.Entity.BeginMoveAsync ( EntityEventArgs  e)
virtual

Is called before a movement is executed (even if the movement will fail).

This method allows the following actions:

  • Change the entity before it is moved
  • Cancel the transaction
  • Initiate another move

Is called in order to make the entity detach itself from its tile.

Typically the desired behavior is achieved by

  • Making the entity pushable using the helper method DetachByPushingAsync(TileEventArgs, Tile)
